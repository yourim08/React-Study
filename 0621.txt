<< SPA >> 
- 하나의 HTML페이지에서, 필요한 데이터나 컴포넌트만 바뀜. 
- 새로고침 필요X 빠름
- 부드러운 사용자 경험
- 모바일 앱처럼 동작하는 웹사이트
- 리엑트는 이걸 만들라고 내놓은 라이브러리

<<컴포넌트>>
조각조각 쪼개놓은것, 함수형 컴포넌트
1. 대문자로 시작
2. props를 통해서 데이터 전달

<<중요한 이유>>
1. 재사용성 : 한번 만들고 여러번 사용
2. 유지보수 : 코드가 기능별로 나뉨 >> 수정, 개선 쉬움
3. 조립식 UI : 컴포넌트를 쌓아서 전체 페이지 구성

<< JSX >> 
JS + XML 

JSX 사용시 주의사항
// 1. class 대신 className 사용
// 2. 모든 태그는 닫혀있어야 함 </>
// 3. 여러 요소를 반환할 때는 반드시 하나의 부모요소로 감싸줘야함
// 4. JSX 는 JS 표현식을 포함할 수 있음. 대신 중괄호로 감싸기
// 5. 주석은 중괄호 안에 작성
// 6. 조건문, 반복문 사용하기 (조건부 랜더링, 반복렌더링)
